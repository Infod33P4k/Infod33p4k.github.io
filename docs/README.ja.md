# 🌐 Infod33p4k.github.io

私の個人ウェブサイトへようこそ。これは、サイバーセキュリティ、ペネトレーションテスト、フルスタック開発に関するブログ、プロジェクト、認定、研究を紹介するデジタルポートフォリオです。

- 📝 攻撃的セキュリティに関する技術ブログ（例：XSS、AD攻撃、トンネリング）
- 💼 実際のレッドチーム活動のためのプロジェクトとツール
- 📚 認定情報と学習の軌跡
- 🔗 外部リソース、参考資料、ガイド
- 🌙 パフォーマンス、可読性、シンプルさを重視した設計

- **フレームワーク**: Astro / Next.js / React
- **スタイリング**: Tailwind CSS / SCSS
- **デプロイ**: GitHub Pages
- **コンテンツ**: Markdown (MDX)
